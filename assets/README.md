# Assets Folder

## Required Images

### Founder Photo
- **Filename**: `founder.jpg`
- **Location**: Place in this folder (`assets/founder.jpg`)
- **Recommended size**: 400x400px to 800x800px
- **Format**: JPG or PNG
- **Usage**: Displayed in About/Founder section

### Instructions
1. Add your founder photo as `founder.jpg` in this folder
2. The website will automatically display it in the About section
3. Ensure the image is professional and high-quality

### Additional Assets (Optional)
- Client logos: `assets/logos/`
- Service images: `assets/services/`
- Case study images: `assets/case-studies/`
